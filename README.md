# Mini Lending Protocol (EVM-Compatible)

A simple decentralized lending prototype built using Solidity + JavaScript.

## Features
- Deposit ETH
- Borrow up to 50% of deposited value
- Repay borrowed funds
- Simple front-end using vanilla JavaScript

## Smart Contract
- Written in Solidity
- Includes deposit, borrow, repay logic
- Emits events for transparency

## Frontend
- Basic HTML + JS interface
- Connects to MetaMask
- Uses ethers.js for interaction

## How to Run
1. Deploy the contract using Hardhat or Remix.
2. Replace the contract address in `frontend/script.js`.
3. Open `index.html` in your browser.
